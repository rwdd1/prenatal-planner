## Getting Started

### Prerequisites

* Node.js
* MongoDB account

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rwdd1/prenatal-planner.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Create a .env file in `/config` and set the required environment variables
